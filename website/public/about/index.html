About CiviForm

Accessing public assistance programs is complicated. Residents must navigate siloed application sites and re-enter sensitive personal information, which can be burdensome and demoralizing. Local program staff and community-based organizations lack the tools necessary to help quickly connect residents to the resources they need most. These problems are only made worse in times of crisis.

CiviForm was built to simplify the way residents find and apply for public assistance programs. When there is no wrong door, government teams can better reach communities in need.

CiviForm is a free, open source software solution for governments that was developed by the City of Seattle with pro bono support from the Google.org Fellowship program. The project is now stewarded by Exygy and is collectively developed and maintained by the governments that use it.

Key features by user

Applicant

Residents seeking assistance.

  • Find public assistance programs in one place
  • Available from a desktop or mobile device
  • Create an account or apply as a guest
  • Apply in your preferred language
  • Apply faster by reusing information across applications
  • Track your application status
  • Discover programs that you may qualify for
  • Understand your potential eligibility before applying

Trusted Intermediary

Third-party orgs that help residents find and apply to public assistance programs.

  • Apply on behalf of a resident
  • Screen for relevant programs on behalf of a resident
  • Manage client applications from the Trusted Intermediary dashboard
  • Track client application status

Program Admin

Staff that review submitted applications and make determinations.

  • Review and make determinations on submitted applications
  • Manage program-specific application status
  • Make internal-facing notes
  • Export application information to review in existing tools

CiviForm Admin

Staff who create and manage program questions and forms.

  • Create, publish, and update program applications
  • Leverage a shared question bank to standardize application forms
  • Show or hide relevant questions and information to applicants
  • Create custom application statuses
  • Specify and validate the content of responses (e.g. addresses)
  • Manage program eligibility criteria for applicant self-screening
  • Export and analyze aggregated information for processing or reporting
  • Manage Program Admin and Trusted Intermediary permissions
